About

Litie Calendar is a modern flat CSS3 calendar design that will help you create a multi-functional dynamic website. The calendars use is completely up to you, it could be used to do anything from creating a day-by-day events timetable, to just displaying the current date for your global users. The latest update has been based on your feedback so please get in touch about what you would like to see next happen to this great calendar.

Customize

If you've got this far your doing great and you may be happy with your current result, you should have beautiful flat red CSS3 calendar staring you in the face. But you might want to display a different colour. Within the code you pasted from the snippet above there ia a key line that will change the way your calendar looks and feels.

					<div id="red-calendarWrapper">

This line of code located on line 1 of the snippet above, it defines which colour your calendar will be at the moment there are six colours to pick from, change this line to any one line below to change the colour of the calender. 

					<div id="red-calendarWrapper">
					<div id="green-calendarWrapper">
					<div id="blue-calendarWrapper">
					<div id="purple-calendarWrapper">
					<div id="orange-calendarWrapper">
					<div id="yellow-calendarWrapper">


You may also want to add your own colour scheme for the calendar to do this you will need to open up the "calendar.css" stylesheet, and at the bottom you will see the following code;

					/* ----- Red Color ----- */
					#red-calendarWrapper #tableHeader{background-color: #f17062;}
					#red-calendarWrapper thead{color: #f17062 !important;}
					#red-calendarWrapper .active{color: #fff; background-color: #f17062;}

Dulipcate this code from either the snippet above or the code found at the bottom of the stylesheet, and change the colour codes to match your needs simply save and your done!
